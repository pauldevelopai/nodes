Nodes

Small AI tools your newsroom runs, owns, and adapts. Each Node installs with a single command and runs on your own computer, on your own data.

Available now

Audience Signal

Reads a Word matrix of your published stories and their Facebook reach + engagement, and shows what your audience actually rewards — by beat, format, headline shape and over time. The signal is engagement rate, not raw reach, so boosting and the algorithm don't fool it.

macOS curl -fsSL https://grounded.developai.co.za/nodes/analytics/mac | bash
Windows irm https://grounded.developai.co.za/nodes/analytics/windows | iex

Paste it into your computer's built-in terminal — nothing to install by hand. Or get the code on GitHub to fork it and make it your own.

How Nodes work
01 · RUN

One command

Install a Node with a single line in your terminal. No accounts, nothing set up by hand. It runs on your machine, on your data.

02 · OWN

Make it yours

The code is open. Download it, edit it, adapt it. Your data stays on your computer — a Node keeps working even if Develop AI disappears tomorrow.

03 · SHARE BACK

Build on each other

Improvements flow back upstream, and other newsrooms build on them — each Node grows from real newsroom experience, not a vendor's roadmap.
