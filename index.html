Nodes

Small AI tools your newsroom runs, owns, and adapts. Each Node installs with a single command and runs on your own computer — or sign in and use it online.

How Nodes work
01 · RUN

One command

Install a Node with a single line in your terminal, or use it online with your Grounded account. It runs on your data.

02 · OWN

Make it yours

The code is open. Download it, edit it, adapt it. Your data stays on your computer — a Node keeps working even if Develop AI disappears tomorrow.

03 · SHARE BACK

Build on each other

Improvements flow back upstream, and other newsrooms build on them — each Node grows from real newsroom experience, not a vendor's roadmap.
